# Skill: gh-fix-ci

## Install

```bash
npx skills add gh-fix-ci
```

## Why it is useful for mo-code

This project spans Flutter and Go, and it will likely grow CI quickly:

- Go builds and tests
- Flutter analysis and builds
- Android packaging checks
- lint or formatting checks

This skill is useful once GitHub Actions becomes part of the normal development path.

## Use this skill when

- a PR check fails
- GitHub Actions logs need inspection
- CI-only regressions appear
- build matrix issues need triage

## Priority

Medium. Not required on day one, but worth installing before CI becomes active.

## Pair with

- `yeet`
- `gh-address-comments`
