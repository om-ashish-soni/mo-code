# Skill: frontend-skill

## Install

```bash
npx skills add frontend-skill
```

## Why it is useful for mo-code

`mo-code` has a strong UI spec, not a generic CRUD app. The UX docs define:

- terminal-style agent output
- provider switcher pills
- task cards
- file tree with git status
- diff preview
- dark design system with specific colors and typography

This skill is useful for converting those specs into intentional Flutter UI work instead of bland defaults.

## Use this skill when

- building `Agent`, `Tasks`, `Files`, or `Config` screens
- implementing terminal output rendering
- implementing streaming UI states
- refining spacing, color, and mobile layout
- translating `docs/UX_MOCKUPS.md` into widgets

## Priority

High. Recommended for any agent doing Flutter or UX-facing work.

## Pair with

- `mo-code-creation`
- `doc` when updating UI specs or handoff notes
