# Skill: gh-address-comments

## Install

```bash
npx skills add gh-address-comments
```

## Why it is useful for mo-code

As soon as this repo starts using pull requests heavily, agents will need a clean review-resolution workflow.

This skill helps with:

- reading actionable review comments
- distinguishing resolved vs unresolved feedback
- implementing selected fixes
- replying with precise context

## Use this skill when

- a PR has requested changes
- review threads need follow-up
- inline comments need code fixes

## Priority

Medium. Useful once the repo has active collaboration and review cycles.

## Pair with

- `yeet`
- `gh-fix-ci`
