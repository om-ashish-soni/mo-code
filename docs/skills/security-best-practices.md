# Skill: security-best-practices

## Install

```bash
npx skills add security-best-practices
```

## Why it is useful for mo-code

After risks are identified, this skill helps agents implement safer defaults in code and config.

It is especially relevant for:

- API key handling
- SSH key generation and storage
- daemon binding and origin checks
- command execution policy
- filesystem scope control
- log redaction

## Use this skill when

- storing secrets in Flutter or Go
- designing config endpoints
- implementing daemon startup and port discovery
- reviewing shell execution behavior
- adding git auth flows

## Priority

High. Recommended for backend and Android integration work.

- `mo-code-creation`
- `security-threat-model`
